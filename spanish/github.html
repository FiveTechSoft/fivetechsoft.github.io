FiveTechSoft en GitHub

Repositorios de código abierto de FiveTech Software, organizados por categoría. Haz clic en cualquier nombre para abrirlo en GitHub.

Librerías GUI

  • fwh — FiveWin — la librería GUI para Harbour/xHarbour en Windows 32/64.
  • fivemac — La librería GUI Cocoa para Harbour en Apple macOS, con el diseño de FiveWin.
  • FiveLinux2 — La librería GUI GTK para Harbour en Linux, con el diseño de FiveWin.
  • FiveLinux3 — FiveLinux sobre GTK 3.0.
  • FiveDroid — GUI Android para Harbour usando Android Studio.
  • FiveOS2 — La librería GUI para CA-Clipper en IBM OS/2.
  • fivedos — GUI estilo FiveWin para xbase en MS-DOS.
  • hbcpp — Una GUI para Harbour basada en C++.
  • fivenet — FiveNet — gestión de clases GUI .NET desde Harbour/xHarbour.
  • FWPPC — FiveWin para Windows Mobile y Windows CE (Pocket PC).

Desarrollo web

  • mod_harbour — Módulo de Apache que ejecuta Harbour como lenguaje web en servidor.
  • fiveweb — Un RAD para la web usando Harbour con sintaxis FiveWin.
  • HbWeb — Un servidor web ligero para Harbour.
  • wsserver — Un servidor WebSocket escrito en Harbour.
  • xcloud — xbase en la nube.
  • harbour_js — Harbour ejecutándose en JavaScript / el navegador.
  • snippets — Fragmentos de código para mod_harbour.
  • fivetechsoft.github.io — Código fuente de esta web.

Compilador Harbour y compilaciones

  • Harbour_builder — Compila todas las variantes de Harbour automáticamente con GitHub Actions.
  • HarbourBuilder — Un entorno de desarrollo visual para Harbour.
  • harbour_and_xharbour_builds — Binarios precompilados de Harbour y xHarbour.
  • xHarbour-builder — Compila automáticamente todas las variantes de xHarbour.
  • harbour_wasm — Harbour compilado con WebAssembly (emscripten).
  • libharbour — Un intérprete de Harbour que corre desde una DLL.
  • harbourllvm — Harbour con backend LLVM — compila xBase sin un compilador C externo.
  • xbase-llvm — Un compilador xbase basado en LLVM.
  • harbourPackages — Paquetes para Harbour.

Bases de datos y ORM

  • ORM — Un ORM para mod_harbour.
  • HbORM — Un ORM para Harbour usando FWH.
  • ORM-for-Dolphin — Un prototipo de ORM usando Harbour y Dolphin.
  • OORDB — Clases de base de datos relacional orientada a objetos para Harbour.
  • tdolphin — Clases de acceso a datos TDolphin.
  • OpenADS — Una versión libre y de código abierto de ADS.
  • dbu — Una utilidad de bases de datos.

IA y aprendizaje automático

  • AIOS — Un sistema operativo de inteligencia artificial.
  • tinyMedical — TinyLlama entrenado con un dataset médico, guardado como GGUF.
  • Harbour_GPT4All — Integración de GPT4All para Harbour.
  • TNeuralNetwork — Una clase TNeuralNetwork para Harbour, portada desde Python.
  • Harbour_simplest_neural_network — La red neuronal más simple implementada en Harbour.
  • transformer — Un modelo transformer implementado en Harbour.
  • quantum-computing-with-Harbour — Simulación de computación cuántica con Harbour.

Móvil

  • fivetouch — Una app tipo dBase para Android e iOS, hecha con Harbour y Qt.
  • fivephone — Una app de teléfono hecha con Harbour.
  • FiveMacArm — FiveMac actualizado para ordenadores Apple M1–M4.

Herramientas e IDEs

  • hbide — Un IDE en modo texto para Harbour que funciona igual en todos los sistemas operativos.
  • EasyReport — Un generador de informes.
  • THaruPDF — Generación de PDF para Harbour basada en HaruPDF.
  • FWH_tools — Herramientas para desarrolladores FiveWin.
  • FWH_samples — Demos y ejemplos compartidos de FiveWin.

Documentación

  • FiveTech_wiki — La wiki de FiveTech / FiveWin.
  • documented_Harbour — El proyecto Harbour totalmente documentado.
  • Harbour_The_definitive_guide — La guía definitiva de Harbour.
  • deepwiki_skill — Un SKILL.md que emula la funcionalidad de deepwiki.
