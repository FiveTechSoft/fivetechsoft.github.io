FiveTechSoft on GitHub

Open source repositories from FiveTech Software, organized by category. Click any name to open it on GitHub.

GUI libraries

  • fwh — FiveWin — the GUI library for Harbour/xHarbour on Windows 32/64.
  • fivemac — The Cocoa GUI library for Harbour on Apple macOS, using the FiveWin design.
  • FiveLinux2 — The GTK GUI library for Harbour on Linux, using the FiveWin design.
  • FiveLinux3 — FiveLinux built on GTK 3.0.
  • FiveDroid — Android GUI for Harbour using Android Studio.
  • FiveOS2 — The GUI library for CA-Clipper on IBM OS/2.
  • fivedos — FiveWin-style GUI for xbase on MS-DOS.
  • hbcpp — A C++ powered GUI for Harbour.
  • fivenet — FiveNet — .NET GUI classes management from Harbour/xHarbour.
  • FWPPC — FiveWin for Windows Mobile and Windows CE (Pocket PC).

Web development

  • mod_harbour — Apache module that runs Harbour as a server-side web language.
  • fiveweb — A RAD for the web using Harbour with FiveWin syntax.
  • HbWeb — A lightweight web server for Harbour.
  • wsserver — A WebSocket server written in Harbour.
  • xcloud — xbase on the cloud.
  • harbour_js — Harbour running in JavaScript / the browser.
  • snippets — mod_harbour code snippets.
  • fivetechsoft.github.io — Source code of this website.

Harbour compiler & builds

  • Harbour_builder — Builds all Harbour flavours automatically with GitHub Actions.
  • HarbourBuilder — A visual development environment for Harbour.
  • harbour_and_xharbour_builds — Prebuilt Harbour and xHarbour binaries.
  • xHarbour-builder — Automatically builds all xHarbour flavours.
  • harbour_wasm — Harbour built with WebAssembly (emscripten).
  • libharbour — A Harbour interpreter running from a DLL.
  • harbourllvm — Harbour with an LLVM backend — compile xBase without an external C compiler.
  • xbase-llvm — An xbase compiler based on LLVM.
  • harbourPackages — Packages for Harbour.

Databases & ORM

  • ORM — An ORM for mod_harbour.
  • HbORM — An ORM for Harbour using FWH.
  • ORM-for-Dolphin — A prototype ORM using Harbour and Dolphin.
  • OORDB — Object-oriented relational database classes for Harbour.
  • tdolphin — TDolphin database access classes.
  • OpenADS — A free and open source version of ADS.
  • dbu — A database utility.

AI & Machine Learning

  • AIOS — An Artificial Intelligence Operating System.
  • tinyMedical — TinyLlama trained with a medical dataset, saved as GGUF.
  • Harbour_GPT4All — GPT4All integration for Harbour.
  • TNeuralNetwork — A TNeuralNetwork class for Harbour, ported from Python.
  • Harbour_simplest_neural_network — The simplest neural network implemented in Harbour.
  • transformer — A transformer model implemented in Harbour.
  • quantum-computing-with-Harbour — Quantum computing simulation with Harbour.

Mobile

  • fivetouch — A dBase-like app for Android and iOS, built with Harbour and Qt.
  • fivephone — A phone app built with Harbour.
  • FiveMacArm — FiveMac updated for Apple M1–M4 computers.

Tools & IDEs

  • hbide — A text-mode IDE for Harbour that works the same across operating systems.
  • EasyReport — A report generator.
  • THaruPDF — PDF generation for Harbour based on HaruPDF.
  • FWH_tools — Tools for FiveWin developers.
  • FWH_samples — Demos and shared FiveWin examples.

Documentation

  • FiveTech_wiki — The FiveTech / FiveWin wiki.
  • documented_Harbour — A fully documented Harbour project.
  • Harbour_The_definitive_guide — The definitive guide to Harbour.
  • deepwiki_skill — A SKILL.md that emulates deepwiki functionality.
